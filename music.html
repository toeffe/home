About the model

ACE-Step 1.5 is an open-source text-to-music generation model. Public project materials describe a hybrid setup: a language model plans structure (metadata, lyrics, captions), and a diffusion transformer (DiT) produces the audio, with alignment trained via intrinsic reinforcement learning.

Details and claims vary by release; see the project repositories for the latest.
